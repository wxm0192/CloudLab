实现NFS文件共享

NFS(Network File System)网络文件系统，是当前主流异构平台共享文件系统之一。

iSCSI实现网络存储

iSCSI（Internet Small Computer System Interface），Internet小型计算机系统接口，又称为IP-SAN，该技术是将现有SCSI接口与以太网络(Ethernet)技术结合

基于 TCP/IP的协议连接iSCSI服务端（Target）和客户端(Initiator)，使得封装后的SCSI数据包可以在通用互联网传输，最终实现iSCSI服务端映射为一个存储空间提供给已连接认证后的客户端。
